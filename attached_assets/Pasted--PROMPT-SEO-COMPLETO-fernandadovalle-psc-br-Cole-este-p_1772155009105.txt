
**Ajuste o `background-color: #1a1a2e`** para a cor de fundo principal do site (provavelmente algo escuro/elegante dado o design). Se tiver o logo "fv" como imagem SVG/PNG, substitua o `<p>fv</p>` por uma tag `<img>`.

No arquivo de inicialização do React (main.tsx, main.jsx ou index.tsx), adicione APÓS o render/mount:

```javascript
const overlay = document.getElementById('loading-overlay');
if (overlay) {
  setTimeout(() => {
    overlay.style.opacity = '0';
    setTimeout(() => { overlay.remove(); }, 500);
  }, 100);
}
```

**Por que isso funciona:** O overlay tem `position:fixed` e `z-index:99999`, então cobre visualmente o conteúdo SEO da div#root. Mas o Google lê o HTML da página inteira — `position:fixed` NÃO esconde conteúdo dos crawlers. O usuário vê uma tela de carregamento elegante, o Google vê todo o conteúdo.

## 6. REDIRECT 301 DO .replit.app

Para evitar conteúdo duplicado, redirecione o domínio Replit para o oficial. No servidor (Express/backend):

```javascript
app.use((req, res, next) => {
  const host = req.hostname || req.headers.host || '';
  if (host.includes('replit.app') || host.includes('repl.co') || host.includes('replit.dev')) {
    return res.redirect(301, `https://fernandadovalle.psc.br${req.originalUrl}`);
  }
  next();
});
```

Se não houver acesso ao backend, garanta que a tag `<link rel="canonical">` aponte para `https://fernandadovalle.psc.br/` em todas as páginas.
